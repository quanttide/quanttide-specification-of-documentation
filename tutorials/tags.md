# 标签属性`tags`

可选标签name和对应verbose_name如下：

## 语言和框架

- `python`: `Python`
- `django`: `Django`
- `flutter`: `Flutter`
- `wechat`: `微信开发`
  
## 领域

- `web_scraping`: `网页爬虫`
- `data_analytics`: `数据分析`
- `database`: `数据库`,
- `cloud_computing`: `云计算`
- `devops`: `DevOps`

## 岗位

- `server`: `服务端开发`
- `client`: `客户端开发`
- `product`: `产品经理'`
- `design`: `交互设计`
